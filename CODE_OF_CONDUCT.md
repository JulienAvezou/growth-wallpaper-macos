# Code of Conduct

GrowthWallpaper is a small, calm, developer-focused project.
We aim to keep the community welcoming, respectful, and low-stress.

This Code of Conduct applies to:

- GitHub issues
- Pull requests
- Discussions
- Theme contributions

---

## Our Standards

We expect everyone to:

- Be respectful and kind
- Assume good intent
- Give constructive feedback
- Respect different skill levels and backgrounds
- Keep discussions focused and technical

---

## Unacceptable Behavior

The following will not be tolerated:

- Harassment, discrimination, or hate speech
- Personal attacks or insults
- Trolling or bad-faith arguments
- Publishing private information without consent

---

## Scope

This Code of Conduct applies within all project spaces and
when representing the project externally.

---

## Enforcement

Maintainers may:

- Remove comments
- Close issues or PRs
- Block or ban contributors if necessary

Enforcement decisions are at the discretion of the project maintainer.

---

## Reporting Issues

If you experience or witness unacceptable behavior, please open a private issue
or contact the maintainer directly.

---

This project follows the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org/),
adapted for a small open-source project.
